mcp-lastfm is a [Model Context Protocol](https://modelcontextprotocol.io) server that gives AI clients read-only access to Last.fm — search the catalogue, look up artists, albums and tracks, browse a listener's scrobble history and weekly charts, and explore what's trending globally, by tag, or by country, all through 41 typed tools.

It authenticates with a single API key, needs no OAuth flow, and never writes to a Last.fm account — no scrobbling, no loving tracks, nothing that mutates a user's data.

## What's here

- [🚀 Getting started](/projects/mcp-lastfm/docs/getting-started) — install, get an API key, and configure your MCP client
- [🔍 Discovery](/projects/mcp-lastfm/docs/discovery) — search, artist/album/track lookups, similarity, corrections, and tags
- [🎧 Listening data](/projects/mcp-lastfm/docs/listening) — a user's scrobbles, top artists/tracks/albums, loved tracks, friends, and weekly charts
- [📊 Charts, tags & geo](/projects/mcp-lastfm/docs/charts-and-geo) — global/tag/country trending charts, tag exploration, and library lookups
