---
title: Installation
description: Add mcp-trakt to Claude Code, Claude Desktop, Cursor, Windsurf, or VSCode.
---

Run `npx @kud/mcp-trakt@latest setup` first — this saves your credentials to the
macOS Keychain so no tokens are needed in any config file. See
[Authentication](/projects/mcp-trakt/docs/authentication) for the full flow.

## Claude Code CLI

```bash
claude mcp add trakt npx -- -y @kud/mcp-trakt@latest
```

## Claude Desktop — macOS

Edit `~/Library/Application Support/Claude/claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "Trakt": {
      "command": "npx",
      "args": ["-y", "@kud/mcp-trakt@latest"]
    }
  }
}
```

Restart Claude Desktop after saving.
